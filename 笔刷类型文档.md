所有的数据类型都为统一格式
因为绘制出来的现在统一为占一个矩阵的大小，因此绘制数据为(x, y) 确定左上角的点，height和width确定矩阵的长宽
{
    type: string, // 笔刷类型
    data: string, // 笔刷数据
    checked: boolean, // 是否被选中
    x: number, // 数据的左上角x坐标
    y: number, // 数据的左上角y坐标
    width: number, // 笔刷数据的宽
    height: number, // 笔刷数据的高
    userId: string, // 用户id
    boardId: string, // 白板id
}
上传数据应该是post的请求
请求为
body {
    data,
    userId,
    boradId,
    Time,
}
